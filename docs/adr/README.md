# Architecture Decision Records

Short, dated records of significant scope / architecture decisions — especially
**deliberate deferrals**, so a "we chose not to build X" is recorded with its
rationale rather than reading as an unexamined gap. Capability gaps that are
*intended to be built* live in [`../ROADMAP.md`](../ROADMAP.md); a decision to
*not* build (or to build one way over another) lives here.
